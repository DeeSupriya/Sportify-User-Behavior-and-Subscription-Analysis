# Spotify User Behaviour Analysis & Subscription Prediction

 Excited to add another project to my growing Data Science journey!

---

##  Project Overview
This project focuses on analyzing Spotify user behavior and building a Machine Learning model to predict whether a user will convert to a premium subscription.

---

##  Objective
To understand user engagement patterns and predict subscription conversion based on user activity, preferences, and interaction with advertisements.

---

##  Dataset Features
- User demographics (age, country)
- Subscription details (type, status)
- Listening behavior (hours per week, skips per day)
- Ad interaction & conversion
- Music preferences (genre, features)
- Device usage

---

## Exploratory Data Analysis (EDA)
Performed data visualization to analyze:
- Subscription type distribution  
- Age vs listening behavior  
- Device usage patterns  
- Genre popularity  
- Impact of ads on subscription conversion  

---

##  Feature Engineering
- Created **Engagement Score** using listening hours and skip behavior  
- Converted categorical data into numerical format  
- Extracted useful features from date

  ---

## Machine Learning Model
- Model Used: **Random Forest Classifier**
- Handled class imbalance using `class_weight='balanced'`
- Train-Test Split: 80% training, 20% testing  

---

## Model Performance
- Accuracy: ~85%  
- Evaluated using:
  - Confusion Matrix  
  - Classification Report  

---

## Key Insights
- Higher listening hours indicate stronger user engagement  
- High skip rates suggest lower satisfaction  
- Ad interaction increases the likelihood of subscription conversion  
- Mobile devices are the most commonly used platform  
- Features like Daily Mix and AI DJ enhance user experience
